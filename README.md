<hr>

<h2>📌 Project Overview</h2>

<p>
This project focuses on analyzing Ola ride booking data using Excel, MySQL, and Power BI.
The dataset was cleaned and prepared in Excel, analyzed using MySQL queries, and visualized through an interactive Power BI dashboard to uncover business insights and operational trends.
</p>

<hr>

<h2>🛠 Tools & Technologies Used</h2>

<ul>
<li>Microsoft Excel – Data Cleaning & Preparation</li>
<li>MySQL – Data Analysis & Querying</li>
<li>Power BI – Dashboard Development & Visualization</li>
</ul>
